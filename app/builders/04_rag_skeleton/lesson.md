# 04. RAG 골격 — 학습 가이드

## 흐름

```
사용자 질문
   ↓
[Retrieve] 임베딩 + 벡터 검색 → top-k 문서
   ↓
[Augment]  프롬프트 = 시스템 + 컨텍스트(인용) + 질문
   ↓
[Generate] LLM 호출
   ↓
답변 + 인용 표시
```

## 본 학습용 단순화

| 영역 | 학습 저장소 | 실무 |
|---|---|---|
| 임베딩 | 키워드 매칭 | OpenAI/Voyage 임베딩 + 코사인 |
| 인덱스 | 메모리 배열 | pgvector / Pinecone / Qdrant |
| 청킹 | 안 함 (이미 짧음) | 500~1000 토큰 청크 |
| 인용 | 텍스트로 노출 | 클릭 가능한 인용 + 원문 링크 |
